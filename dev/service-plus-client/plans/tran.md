# Access control fine tuning
- short names for roles: Man, Tech, Rec for Manager, Technician, Receptionist
- Use disabled + tooltip instead of hide/show
- Switch to Admin mode icon is only visible to Admin. If admin user, he can do everything. No restrictions on Admin.
- tech mode
    - disabled for Receipts, Opening Jobs, Accounts Posting in the jobs menu.
    - disabled Masters, Configurations, Admin menu.
- Rec Mode
    - disabled admin, configurations menu item.
- Man Mode
    - can do everything. Nothing is disabled.
    - only "Switch to Admin" icon is disabled to Man.
